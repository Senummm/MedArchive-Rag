# Test Document: Diabetes Management Summary

## Overview

This is a simplified test document for validating the ingestion pipeline.

## Treatment Protocol

### Oral Medications

| Medication | Starting Dose | Maximum Dose |
|------------|---------------|--------------|
| Metformin | 500mg daily | 2000mg daily |
| Glipizide | 5mg daily | 40mg daily |

### Monitoring

- Check HbA1c every 3 months
- Target HbA1c <7% for most patients
- Adjust medications based on response

## Patient Education

Key points to emphasize:
1. Diet modification
2. Regular exercise
3. Medication adherence
4. Blood glucose monitoring
